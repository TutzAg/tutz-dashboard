[TUTZ] Dashboard é um plugin versátil que adiciona funcionalidades essenciais e práticas ao WordPress, como botão do WhatsApp, minificação de CSS e JS, além de integrações com o WooCommerce.

== Descrição ==
O [TUTZ] Dashboard foi projetado para ajudar administradores de sites WordPress a gerenciar e otimizar seu site com facilidade. Ele inclui:

* Adição de um botão do WhatsApp no conteúdo das páginas/posts.
* Minificação de arquivos CSS e JavaScript para melhor desempenho.
* Configurações práticas e intuitivas diretamente no painel administrativo.
* Integração com WooCommerce para funcionalidades adicionais, como redirecionamento direto para o checkout.

== Instalação ==
1. Faça o upload da pasta do plugin para o diretório `/wp-content/plugins/` ou instale diretamente via painel do WordPress.
2. Ative o plugin através do menu "Plugins" no WordPress.
3. Configure as opções no menu "TUTZ Dashboard" no painel administrativo.

== FAQ ==

= Este plugin é compatível com o WooCommerce? =
Sim, possui funcionalidades específicas para lojas WooCommerce.

= Como configuro o botão do WhatsApp? =
Basta adicionar o número no formato internacional no painel administrativo em "TUTZ Dashboard".

= O que acontece se eu ativar a minificação de CSS e JS? =
Os arquivos serão otimizados automaticamente para melhorar a velocidade de carregamento.
